# Car Black Box

## Overview
Car Black Box is an Embedded C project developed using the PIC16F877A microcontroller. The system records important vehicle information to assist in monitoring and diagnostics.

## Features
- Event logging
- Real-Time Clock (DS1307)
- EEPROM data storage
- Matrix keypad interface
- Character LCD display
- UART communication
- I2C communication

## Hardware
- PIC16F877A
- 16x2 Character LCD
- DS1307 RTC
- External EEPROM
- Matrix Keypad

## Software
- Embedded C
- MPLAB X IDE
- XC8 Compiler

## Project Structure
- main.c
- adc.c
- clcd.c
- uart.c
- i2c.c
- ds1307.c
- eeprom.c
- matrix_keypad.c
